FreeCodeCamp API Basejump: Image Search Abstraction Layer

User stories:
    1) I can get the image URLs, alt text, and page urls for a set of images relating to a given search string.
    2) I can paginate through the responses by adding a ?offset=2 parameter to the URL.
    3) I can get a list of the most recently submitted search strings.

Example usage:

Example Form for searching image URLs, alt text, and page urls:

https://jeuscortez-image-search-abstraction-layer.glitch.me/api/imagesearch/pokemon?offset=2

Example Form for getting list of the most recently submitted search strings

https://jeuscortez-image-search-abstraction-layer.glitch.me/api/latest/imagesearch
